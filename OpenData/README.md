# 原始資料集
尚未處理的原始資料,內有台灣氣候及台灣疾病的資料分類
### 台灣氣候
主要資料內容為台灣各監測站資料和台南歷年每月均溫與降水量資料，主要取用欄位為「年份」,「縣市」,「平均溫度」,「降水量」
* #### 台灣1897-1945平均溫度
台灣1897-1945每個月的平均溫度目的是要找出台灣全球暖化的證據

資料來源：[歷年分月平均氣溫](http://twstudy.iis.sinica.edu.tw/twstatistic50/Climate.htm)

* #### 台灣各偵測站2008至2016觀測資料

資料來源：[一年觀測資料-局屬地面測站觀測資料](http://opendata.cwb.gov.tw/catalog?group=c&dataid=B0024-002)

* #### 台南歷年每月均溫及降水量

從台灣各偵測站2008至2016觀測資料整理出來的資料
資料範圍為2012至2016年

### 疾病
台灣目前與環境相關並常發生的傳染病，目前的資料只做登革熱的分析
主要取用欄位有「確定病名」、「發病年份」、「發病月份」、「縣市」、「確定病例數」
* #### 2003至2016登革熱各地案例

資料來源：[地區年齡性別統計表-登革熱](http://data.gov.tw/node/6879)
